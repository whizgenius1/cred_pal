# CredPal Assessment
An assessment for credpal interview process


# Commands
- **To generate assets**: dart run build_runner build --delete-conflicting-outputs

- **To build android apk**: flutter build apk

- **To build android bundle**: fvm flutter build appbundle

- **To build android iOS**: fvm flutter build ios

- **To build android ipa**: flutter build ipa

- **To build android web**: flutter build web

- **To build android web assebly**: flutter build web --wasm



## Prerequisites

**Flutter SDK**: Install Flutter
**Dart**: Comes with Flutter SDK.
A code editor like VS Code or Android Studio.

---

## Installation

1 **Clone the repository**
- [Clone the Application](https://github.com/your-repo/ivorypay-mobile-wallet-app.git)
- ``git clone https://github.com/your-repo/ivorypay-mobile-wallet-app.git``

2 **Navigate to the project directory**
- ``cd cred_pal``

3 **Install dependencies**
- ``flutter pub get``

4 **Generate Build Dependencies**
- ``dart run build_runner build --delete-conflicting-outputs``


Since this is an assesment base on just UI and not core functionality. a makefile is not provided.
